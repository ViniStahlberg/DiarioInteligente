# SmartDiary - O diário inteligente que entende o seu momento 🧠✨

O **SmartDiary** é uma aplicação Android nativa desenvolvida como projeto final para o curso de Tecnologia em Análise e Desenvolvimento de Sistemas (IFSP). O objetivo é oferecer uma ferramenta de autoconhecimento que vai além do registro de texto, capturando o contexto ambiental e físico do usuário para identificar padrões de bem-estar.

## 🚀 Funcionalidades Principais

- **Registro Multimídia:** Adicione textos, fotos (CameraX) e notas de voz (SpeechRecognizer) aos seus momentos.
- **Contexto Automático:** Captura a luminosidade do ambiente (sensor de LUX) para entender como o ambiente afeta seu humor.
- **Acompanhamento de Humor:** Visualização clara da evolução emocional da semana através de gráficos interativos.
- **Segurança Biométrica:** Acesso protegido por digital ou reconhecimento facial (BiometricPrompt).
- **Sincronização em Nuvem:** Seus dados seguros e acessíveis via Firebase.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Kotlin
- **Arquitetura:** MVVM (Model-View-ViewModel)
- **Banco de Dados & Auth:** Firebase (Firestore & Authentication)
- **APIs de Hardware:** 
  - `CameraX` para captura de fotos.
  - `SpeechToText` para transcrição de voz.
  - `SensorManager` para leitura de luminosidade.
  - `Biometric Library` para segurança nativa.

## 📈 Mercado e Diferenciais

O SmartDiary foca em uma dor real: **70% das pessoas não conseguem identificar padrões no próprio humor**. Enquanto concorrentes focam apenas em entrada manual, nós utilizamos os sensores do smartphone para prover insights contextuais automáticos.
